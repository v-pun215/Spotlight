# 🌍 Spotlight

**Spotlight** is a modern tourism discovery website focused on **Delhi**, letting you explore trending places, search hidden gems, and personalize your experience.

Built with HTML, CSS, JS on the frontend and Flask + Express on the backend.

---

## 🚀 Features

- Explore top destinations in **Delhi** with dynamic cards  
- Real-time search powered by Foursquare & Wikipedia  
- User signup/login with token-based auth  
- Profile setup with name & username  
- Clean, responsive UI

---

## 🛠 Tech Stack

- **Frontend**: HTML, CSS, JS  
- **Backend**: Flask (auth), Express.js (places)  
- **Database**: MongoDB  
- **APIs**: Foursquare, Wikipedia
